# 随笔：直到死亡把我们分开

> 无论贫穷还是富有，健康还是疾病，相爱相依，不离不弃，直到死亡把我们分开。

这是一段非常经典的、真挚感人的誓词，我们都知道这一段话的意思。

害怕承诺是自然的，我其实挺欣赏一个人因为有责任感才害怕承诺的。「这个杀手不太冷｜里面，玛蒂尔达抱着两盒牛奶，强忍泪水经过刚被全家灭口的自己家门，敲 Leon 家的门。Leon 在锁孔看到了一切，他犹豫了很久要不要给一个只见过两面的女孩开门。女孩住进他家的第一晚，他甚至犹豫要不要一枪崩了这个原本和自己毫无关系的女孩。

愿意许下承诺是勇敢的，但我在今年看到了很多案例，覆盖多个社会阶层身份、在关系里的位置，我终于发现一个一直被忽略的事实：人的意愿与能力完全是两件事。很多人是心中有爱、嘴上也表达爱，但行动上根本做不到的。

这样说的目的不是苛责任何一个人，而是在我充分理解了他人深入骨髓的弱之后，不禁照见自己。如果把上述诺言修改为一段对自己的诺言，我可以做到吗？

> 无论贫穷还是富有，健康还是疾病，我都尊重自己的感受、捍卫自己的尊严、为自己真正想做的事情努力、永远不放弃自己、爱自己，直到死亡把我和我自己分开。

—— 我能做到吗？

对自己的要求和爱应该充分到有剩余，才有能力不虚假自尊，从而能听取他人意见；有能力爱护弱者；有能力爱及他人甚至做出承诺；也有能力建立自己的人际交往框架，进而筛选出和自己价值观、愿景相近的人共同相处。

这一切的前提是我足够爱我自己。

而这种逻辑并不限于爱情，也可以用来理解有些有毒的亲子关系：一边觉得孩子是“白眼狼”，自己牺牲很多；一边觉得对方一味付出，关系特别扭曲。其实，如果双方都把自己的成长优先级放在管控他人之前，问题就迎刃而解。

「主体性」是今年互联网上特别流行的一个词语，主体性的觉醒是一个人成长的真正标志。

成长是一个更大的话题，这篇文章只是想要提醒自己爱是需要能力的，而在自己身上培养这些能力是唯一可行的也是稳赚不赔的。

人最本质的需求还是爱和尊重，只是得到这一切，各有各的功课。
